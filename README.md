言葉を自分の心に留めておかず、この場所に置いていくための「感情のデトックス・アプリ」です。

​[アプリを体験する → https://shunrai.vercel.app
​
⚡️ コンセプト
​「思考と行動を軽くする」をテーマに開発されました。
私たちの言葉は、色んな思いからうまれます。ある時は「さみしさや不安」からかもしれません。
​このアプリは、入力された言葉を離すお手伝いをさせていただきます。
​✨ 主な機能
​言葉の浄化: 言葉を入力し、深呼吸とともに手放すプロセスを体験。
​

​　使用技術
​Frontend: React / Next.js (App Router)
​Styling: Tailwind CSS
​Animations: CSS Keyframes / Framer Motion (予定)
​Architecture: シンプルかつ軽量なユーザー体験を追求
​ 今後のロードマップ
​現在はシンプルなフロントエンド実装ですが、今後は以下のアップデートを予定しています。
​AI (Dify) 連携: 入力された言葉を解析し、よりパーソナライズされた「相手の心の叫び」を生成。
​Antigravity 移行: より軽快でスケーラブルな基盤への最適化。
​マルチデバイス対応: いつでも、どこでも心を軽くできるモバイル体験の強化。

​🖋 開発者の想い
​現在、塾講師として活動しながら「人の心を軽くする」ことを目標にアプリ開発をしています。
複雑すぎる現代において、シンプルに「生きるのを楽にする」ツールを届けたい。その第一歩が、この『春来』です。
